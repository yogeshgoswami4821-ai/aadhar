This project solves the challenge of **"Unlocking Societal Trends"** using Aadhaar data.

### Key Insights Provided:
* **Saturation Gaps**: Identifying districts with low enrolment for targeted camps.
* **Gender Parity**: Spotting regions where female enrolment lags behind.
* **Migration Patterns**: Analyzing address updates to map internal migration trends.

### Tech Stack:
* **Backend**: Flask (Python) + Pandas for Data Crunching.
* **Frontend**: HTML5, CSS3 (Tailwind-inspired), Chart.js.
* **Deployment**: Render (Backend) & GitHub Pages (Frontend).
